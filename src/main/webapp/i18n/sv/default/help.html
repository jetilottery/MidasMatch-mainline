Midas Match

Låt Midas Match förvandla dina matchningar till gyllene rikedomar! När du avslöjar tre matchande symboler vinner du ett pris. Håll utkik efter de särskilda guldmynten som kan ge dig ett bonusspel: aktivera hjulbonusen och snurra för att vinna, eller aktivera myntbonusen och välj mynt för att få priser!

Så här satsar du

Lottkostnad

Visar den aktuella lottkostnaden.

Tryck på vänsterpilen (–) för att minska lottkostnaden.

Tryck på högerpilen (+) för att öka lottkostnaden.

Tryck på KÖP för att dra lottkostnaden från saldot och börja spela.

Så här spelar du

Tryck på varje ruta för att avslöja symboler. Avslöja 3 matchande symboler för att vinna motsvarande pris.

Avslöja myntbonus-guldmyntet för att aktivera myntbonusen. Välj mynt för att avslöja priser. När du avslöjar INKASSERA avslutas bonusen.

Avslöja hjulbonus-guldmyntet för att aktivera hjulbonusen. Snurra hjulet. När hjulet stannar vinner du det markerade beloppet.

Alla eventuella bonusspel spelas efter att alla symboler i grundspelet har avslöjats.

Spelet avslutas när alla symboler har avslöjats och alla tilldelade bonuspel är färdiga.

Avfärda vinstmeddelandet och se spelets utfall genom att trycka på meddelandet.

Avslöja alla

Tryck på AVSLÖJA ALLA för att spela klart spelet automatiskt. Spelet fortsätter tills alla val har gjorts och alla eventuella priser har delats ut.

Regler

Missbruk eller funktionsfel ogiltigförklarar alla utbetalningar och spel.

Bonusvinster läggs till det totala prisvärdet.

Vinster visas i vald valuta.

Val som gjorts av spelare under spelets gång påverkar inte spelets utfall.
