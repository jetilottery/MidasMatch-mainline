Midas Match

Die goldene Berührung von Midas Match verwandelt Ihre Übereinstimmungen in Reichtümer! Gewinnen Sie einen Preis, wenn Sie drei übereinstimmende Symbole aufdecken. Achten Sie auf die besonderen Goldmünzen, die Sie ins Bonusspiel bringen: Lösen Sie den Rad-Bonus aus, in dem Sie das Rad für einen Gewinn drehen, oder lösen Sie den Münzbonus aus, um Münzen für Preise auszuwählen!

So wird gespielt

Lospreis

Zeigt den aktuellen Lospreis an.

Drücken Sie auf den linken Pfeil (–), um den Lospreis zu verringern.

Drücken Sie auf den rechten Pfeil (+), um den Lospreis zu erhöhen.

Drücken Sie auf KAUFEN, um den Lospreis vom Guthaben abzuziehen und das Spiel zu spielen.

Spielanleitung

Drücken Sie auf jede Kachel, um Symbole aufzudeckn. Decken Sie 3 übereinstimmende Symbole auf, um einen gewinnen Sie den entsprechenden Preis.

Decken Sie die Münzbonus-Goldmünze auf, um den Münzbonus auszulösen. Wählen Sie Münzen aus, um Preise aufzudecken. Decken Sie KASSIEREN auf, um den Bonus zu beenden.

Decken Sie die Rad-Bonus-Goldmünze auf, um den Rad-Bonus auszulösen. Drehen Sie das Rad und gewinnen Sie den markierten Betrag, wenn das Rad anhält.

Alle vergebenen Bonusspiele werden gespielt, nachdem alle Symbole im Basisspiel aufgedeckt wurden.

Das Spiel endet, wenn alle Symbole aufgedeckt wurden und alle vergebenen Bonusspiele abgeschlossen sind.

Verwerfen Sie die Gewinnnachricht und schauen Sie sich das Spielergebnis an, indem Sie auf die entsprechende Nachricht drücken.

Alle aufdecken

Drücken Sie ALLE AUFDECKEN, um das Spiel automatisch zu Ende zu spielen. Das Spiel wird fortgesetzt, bis alle Auswahlen getroffen und die entsprechenden Preise vergeben wurden.

Regeln

Missbrauch oder Fehlfunktionen annullieren alle Zahlungen und Spiele.

Bonusgewinne werden zum Gesamtgewinnbetrag addiert.

Gewinne werden in der ausgewählten Währung angezeigt.

Spielerentscheidungen, die während des Spiels getroffen werden, haben keinen Einfluss auf das Spielergebnis.
